    HTML

    HTML（超文本标记语言）是用于创建网页的标准标记语言。它构成了网页的骨架，定义了网页的结构和内容。通过使用HTML标签，开发者可以创建标题、段落、链接、图片、列表等各种网页元素。HTML标签不仅描述了网页的内容，还定义了这些内容应该如何被浏览器解析和显示。HTML文档通常以.html或.htm为扩展名，并且可以通过任何支持HTML的浏览器来打开和查看。由于其简单易懂的结构和广泛的浏览器支持，HTML成为了开发网页的基础语言，无论是个人网站、企业官网还是复杂的网络应用，都离不开HTML的支持。

    CSS

    CSS（层叠样式表）是用于描述HTML文档样式的标记语言。它使得网页设计师能够精确地控制网页上元素的外观和布局，包括颜色、字体、大小、间距等。CSS通过将样式信息与HTML内容分离，提高了网页的可维护性和可重用性。通过使用CSS，设计师可以创建出视觉上吸引人的网页，并且能够在不同的设备和屏幕尺寸上实现一致的显示效果。CSS还提供了许多高级特性，如动画、渐变、变换等，使得网页更具动态性和互动性。通过外部链接、内部样式或行内样式，CSS可以轻松地应用到HTML文档中，为网页带来丰富的视觉体验。

    javascript

    JS，全称JavaScript，是一种流行的轻量级、解释型编程语言。以下是关于JS的简要介绍：
    定义与起源： JavaScript是一种基于原型的、多范式的动态脚本语言，支持面向对象、命令式、声明式、函数式编程范式。 它最初由Netscape的Brendan Eich设计，并在1995年作为网景导航者浏览器的一部分首次实现。
    特点： 解释型：代码不需要预编译，可以直接在浏览器中运行。 基于对象：可以创建和操作对象。 弱类型：变量类型不固定，可以动态改变。 动态性：可以实时响应用户操作，如点击、滚动等。 跨平台：不依赖特定操作系统，只要有支持JavaScript的浏览器即可运行。
